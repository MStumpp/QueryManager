Create QueryManager instance

Allocate an instance of the QueryManager in the following way.

QueryManager *manager = [QueryManager instance];

Configure QueryManager instance

You may set the maximum number of concurrent connections for all queues.

[manager setNumberOfMaxConcurrentConnections:10];
  • numberMaxConcurrent: Maximum number of concurrent connections over all queues. Lowest value is 0. Use tUnlimitedConcurrentConnections for unlimited number of connections.

Create Queue instance

Allocate an instance of Queue in the following way.

Queue *queue = [[Queue instance] initWithIdentifier:@"queue 1" queueMaxSize:tUnlimitedQueueSize numberMaxConcurrent:tUnlimitedConcurrentConnections processOrder:tfifo finishHigherPrioFirst:TRUE pauseWhileLoading:TRUE cancelWhileLoading:TRUE];
  • identifier: A unique identifier per queue.
  • queueMaxSize: Maximum size of a queue, meaning number of Query instances in state initial, loading, pausing or paused (Query instances in state loaded, cancelling or cancelled are not count towards this metric). Use tUnlimitedQueueSize for unlimited number of Query instances.
  • numberMaxConcurrent: Maximum number of concurrent connections over all queues. Lowest value is 0. Use tUnlimitedConcurrentConnections for unlimited number of connections.
  • processOrder: Order of how Query instances will be processed. Use tfifo for first in first out or tlifo for last in first out.
  • finishHigherPrioFirst:
  • pauseWhileLoading: Pauses all Query instances which come after the added one, from the other queues as well as the same queue.
  • cancelWhileLoading: Cancels all Query instances which come after the added one, from the other queues as well as the same queue.

Add Queue instance to QueryManager

Register a Queue instance and set a priority.

queue = [manager addQueue:queue prio:0];

Every QueryManager can have only one instance of Queue of the same priority.

Lowest value for priority is 0 (highest priority).

Create Query instance

QueryTicket *ticket = [[queue addQuery:[ConcreteQuery instance] andPrio:0] process:^() {

}];

QueryTicket *ticket = [[ConcreteQuery instanceWithQueue:queue andPrio:0] process:^() {

}];
